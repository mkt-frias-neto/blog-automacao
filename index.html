Como funciona

Esta ferramenta gera matérias completas e otimizadas para SEO usando Claude, com hiperlinks e UTMs corretos para o site da Frias Neto.

Após gerar, você recebe o HTML pronto para colar diretamente no editor WordPress.

Selecione o tema ao lado, informe sua chave de API do Claude e clique em Gerar matéria.

Como colar no WordPress

  1. Abra o WordPress e clique em Posts → Adicionar novo
  2. Clique no botão + e adicione um bloco HTML Personalizado
  3. Cole o conteúdo abaixo dentro do bloco
  4. Adicione imagem de destaque, tags e categoria
  5. Clique em Publicar

Passo a passo para publicar

  1. Gere a matéria usando esta ferramenta
  2. Copie o HTML na aba HTML WordPress
  3. Copie os metadados na aba SEO & Meta
  4. Acesse blog.friasneto.com.br/wp-admin
  5. Clique em Posts → Adicionar novo
  6. Clique em + → HTML Personalizado e cole o HTML
  7. Preencha Categoria, Tags e Imagem destacada no painel lateral
  8. Cole o meta title e meta description no Yoast SEO ou RankMath
  9. Defina o slug na URL do post
  10. Clique em Publicar

Plugin SEO recomendado

Instale o Yoast SEO ou RankMath — ambos gratuitos. Eles adicionam campos de meta title e meta description diretamente na tela de edição do post.

Imagem de destaque

Gratuito: Unsplash (unsplash.com) — busque pelo tema sugerido, baixe e faça upload no WordPress

Com IA: Canva, Adobe Firefly ou Midjourney para imagens exclusivas
